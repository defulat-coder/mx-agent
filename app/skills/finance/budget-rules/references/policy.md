# 马喜公司预算管理制度

## 一、预算编制

1. 每年 11 月启动次年预算编制
2. 各部门提交年度预算申请
3. 财务部汇总审核
4. CEO 批准后执行

## 二、预算科目

| 科目 | 说明 |
|------|------|
| 差旅费 | 出差交通、住宿、餐费 |
| 办公费 | 办公用品、设备采购 |
| 招待费 | 客户招待、商务活动 |
| 培训费 | 员工培训、外部课程 |
| 市场费 | 推广、广告、活动 |
| 其他 | 未归类费用 |

## 三、预算执行

1. 各部门按批准预算执行
2. 财务部按月出具预算执行报告
3. 预算执行率超 80% 时系统自动预警
4. 超 90% 须部门负责人确认后方可继续支出

## 四、预算调整

1. 预算不足时可申请追加
2. 追加申请须说明原因和金额
3. 审批流程：部门负责人 → 财务总监 → CEO
4. 单次追加不超过原预算 20% 的，财务总监可审批
5. 超过 20% 须 CEO 审批

## 五、超支管理

1. 未经审批不得超支
2. 紧急超支须事后 3 个工作日内补办审批
3. 未补办审批的超支由个人承担
4. 连续 2 个季度超支的部门，下季度预算冻结审查

## 六、预算监控

1. 财务部每月发布各部门预算执行报告
2. 部门负责人可随时查询本部门预算余额
3. 每季度召开预算分析会议
