# 假期制度咨询

当员工询问以下内容时使用本 Skill：
- 各类假期天数及计算规则
- 请假流程与审批权限
- 假期余额问题
- 销假与假期延期
- 特殊假期（婚假、产假等）所需材料

## 使用方式

1. 通过 `get_skill_reference("leave", "policy.md")` 获取完整假期制度
2. 如需计算员工年假天数，使用 `get_skill_script("leave", "calc_annual_leave.py")`
3. 涉及具体假期余额时，调用假期余额查询 Tool
4. 涉及请假申请时，引导员工使用审批系统并提供链接

## 注意事项

- 年假天数与司龄和社会工龄均相关，需综合计算
- 病假需提供医院证明，3 天以上需二级以上医院证明
- 产假/陪产假天数因城市政策不同而有差异
