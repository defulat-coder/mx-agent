# 假期管理制度

## 1. 年假

### 1.1 年假天数

按**累计社会工龄**计算法定年假：

| 累计工龄 | 法定年假 |
|---------|---------|
| 1 年 ≤ 工龄 < 10 年 | 5 天 |
| 10 年 ≤ 工龄 < 20 年 | 10 天 |
| 工龄 ≥ 20 年 | 15 天 |

公司额外福利年假（按司龄）：

| 司龄 | 额外年假 |
|------|---------|
| 满 1 年 | +1 天 |
| 满 3 年 | +2 天 |
| 满 5 年 | +3 天 |
| 满 10 年 | +5 天 |

### 1.2 年假规则

- 入职当年按剩余日历天数折算，不足 1 天不计
- 年假须在当年 12 月 31 日前休完，**最多可顺延至次年 3 月 31 日**
- 过期未休的年假清零，不折算工资
- 连续请年假超过 5 天需提前 **2 周** 申请

## 2. 病假

| 项目 | 规则 |
|------|------|
| 天数 | 医疗期按工龄确定（3-24 个月不等） |
| 薪资 | 病假工资 = 基本工资 × 80%（不低于当地最低工资标准） |
| 材料 | ≤3 天：医院病假条；>3 天：二级以上医院病假条 + 病历 |
| 审批 | ≤3 天：直属主管；>3 天：部门总监；>5 天：HR 审核 |

## 3. 事假

| 项目 | 规则 |
|------|------|
| 天数 | 全年累计不超过 **15 天** |
| 薪资 | 事假期间 **无薪** |
| 审批 | ≤1 天：直属主管；>1 天：部门总监；>3 天：HR 审核 |

## 4. 婚假

| 项目 | 规则 |
|------|------|
| 天数 | **3 天**（法定）+ 各地额外奖励假 |
| 薪资 | 全额发放 |
| 材料 | 结婚证 |
| 有效期 | 自领证之日起 **1 年内** 休完 |
| 说明 | 须一次性连续休完，含周末及法定节假日 |

## 5. 产假 / 陪产假

### 5.1 产假（女员工）

| 情形 | 天数 |
|------|------|
| 正常生育 | 98 天 + 各地奖励假（上海 60 天） |
| 难产 | +15 天 |
| 多胞胎 | 每多 1 胎 +15 天 |

- 薪资：按生育津贴标准发放，不足部分公司补差
- 产检假：按医院建议次数，每次半天

### 5.2 陪产假（男员工）

| 项目 | 规则 |
|------|------|
| 天数 | **10 天**（各地政策不同） |
| 薪资 | 全额发放 |
| 有效期 | 自配偶生育之日起 **1 个月内** 休完 |

## 6. 丧假

| 关系 | 天数 |
|------|------|
| 父母、配偶、子女 | **3 天** |
| 祖父母、外祖父母、配偶父母 | **3 天** |

- 外地料理后事的可额外给予路程假（凭票据）
- 薪资全额发放

## 7. 调休假

- 来源：工作日或周末加班产生的调休额度
- 有效期：自加班日起 **6 个月**
- 使用：按正常请假流程申请，优先使用即将过期的调休

## 8. 请假流程

```
员工提交申请 → 直属主管审批 → (如需)部门总监审批 → (如需)HR审核 → 生效
```

### 审批权限汇总

| 假期时长 | 审批层级 |
|---------|---------|
| ≤ 1 天 | 直属主管 |
| 1-3 天 | 部门总监 |
| > 3 天 | 部门总监 + HR |
| > 5 天 | 部门总监 + HRBP + HR 总监 |

### 注意事项

- 请假须提前申请，特殊紧急情况可事后 **1 个工作日内** 补申请
- 病假当天无法提前申请的，须电话/微信告知主管，事后 **3 个工作日内** 补交病假条
- 未经审批擅自休假视为旷工
