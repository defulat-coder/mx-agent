# 报销管理制度

## 1. 报销类型

| 类型 | 说明 |
|------|------|
| 差旅报销 | 因公出差产生的交通、住宿、餐饮费用 |
| 业务招待 | 因业务需要的客户招待费用 |
| 办公用品 | 部门采购的办公耗材 |
| 培训费用 | 经批准的外部培训/考试费用 |
| 通讯费 | 超出补贴部分的因公通讯费 |
| 其他 | 其他经审批的因公支出 |

## 2. 差旅报销标准

### 2.1 交通标准

| 职级 | 飞机 | 高铁/动车 | 其他 |
|------|------|---------|------|
| 总监及以上 | 经济舱 | 一等座 | 实报实销 |
| 经理/高级 | 经济舱 | 二等座 | 实报实销 |
| 普通员工 | 经济舱（4小时以上航程） | 二等座 | 实报实销 |

- 4 小时以内优先选择高铁
- 市内交通优先使用公共交通，打车需说明原因

### 2.2 住宿标准

| 城市类别 | 总监及以上 | 经理/高级 | 普通员工 |
|---------|-----------|---------|---------|
| 一线城市 | ≤ 800 元/晚 | ≤ 500 元/晚 | ≤ 400 元/晚 |
| 二线城市 | ≤ 600 元/晚 | ≤ 400 元/晚 | ≤ 300 元/晚 |
| 其他城市 | ≤ 500 元/晚 | ≤ 350 元/晚 | ≤ 250 元/晚 |

- 一线城市：北京、上海、广州、深圳
- 二线城市：杭州、成都、南京、武汉、苏州等省会/计划单列市
- 超标需提前申请并由部门总监审批

### 2.3 餐饮补贴（出差期间）

| 餐次 | 标准 |
|------|------|
| 早餐 | 30 元 |
| 午餐 | 50 元 |
| 晚餐 | 50 元 |
| 合计 | 130 元/天 |

- 如出差有客户招待餐，当餐不再享受餐补
- 实报实销，不足不补

## 3. 业务招待标准

| 接待级别 | 人均标准 | 审批人 |
|---------|---------|--------|
| 普通业务接待 | ≤ 200 元/人 | 部门经理 |
| 重要客户接待 | ≤ 500 元/人 | 部门总监 |
| 高端商务接待 | ≤ 800 元/人 | VP 及以上 |

- 需提前填写《招待申请单》
- 报销时附参与人员名单（含客户信息）

## 4. 发票要求

### 4.1 基本要求

- 必须为**增值税普通发票**或**增值税专用发票**
- 抬头：公司全称（马喜科技有限公司）
- 税号：须正确无误
- 发票日期：与实际消费日期一致

### 4.2 不予报销的情况

- 非正规发票（收据、白条等）
- 发票抬头为个人
- 发票信息有涂改
- 超过报销时效（消费后 **3 个月** 内未报销）
- 发票金额与实际消费不符

## 5. 报销流程

```
员工提交报销申请（OA 系统）
  ↓
附上发票照片/扫描件
  ↓
直属主管审批
  ↓
（金额 > 2000 元）部门总监审批
  ↓
（金额 > 10000 元）财务总监审批
  ↓
财务审核发票合规性
  ↓
打款至工资卡
```

### 审批时效

| 环节 | 时限 |
|------|------|
| 主管审批 | 3 个工作日 |
| 总监审批 | 3 个工作日 |
| 财务审核 | 5 个工作日 |
| 打款 | 审核通过后 **5 个工作日** |

## 6. 注意事项

- 报销须在费用发生后 **3 个月内** 提交，逾期不予受理
- 跨年度报销截止至次年 **1 月 15 日**
- 大额支出（> 5,000 元）建议提前走审批流程
- 团建费用走团建预算，不走个人报销
- 禁止虚假报销，一经发现按严重违纪处理
