# 离职管理制度

## 1. 离职类型

| 类型 | 说明 |
|------|------|
| 主动离职 | 员工因个人原因提出辞职 |
| 协商解除 | 双方协商一致解除劳动合同 |
| 合同到期不续签 | 合同到期，任一方不续签 |
| 辞退 | 因严重违纪或不胜任岗位 |

## 2. 提前通知期

| 状态 | 提前通知期 |
|------|-----------|
| 试用期 | **3 天** |
| 正式员工 | **30 天**（一个月） |
| 高管/特殊岗位 | **60-90 天**（以合同约定为准） |

- 通知期自离职申请审批通过之日起计算
- 公司有权要求员工在通知期内正常出勤完成交接
- 经协商一致，可缩短通知期

## 3. 离职流程

```
员工提交离职申请（HR 系统）
  ↓
直属主管面谈（1-3 个工作日内）
  ↓
主管审批 → 部门总监审批 → HR 审批
  ↓
确定最后工作日
  ↓
工作交接（主管指定交接人）
  ↓
资产归还（电脑、工牌、钥匙等）
  ↓
IT 账号注销
  ↓
财务结算
  ↓
HR 出具离职证明
```

## 4. 工作交接

### 4.1 交接内容

| 类别 | 交接项目 |
|------|---------|
| 工作事项 | 进行中的项目、待办事项、工作文档 |
| 客户/供应商 | 对接关系移交、联系方式整理 |
| 系统权限 | 各系统账号、密码（加密移交） |
| 资产物品 | 电脑、显示器、工牌、钥匙、文具等 |
| 文档资料 | 纸质文件、电子文档归档 |

### 4.2 交接要求

- 填写《工作交接清单》（HR 系统下载模板）
- 交接人由直属主管指定
- 交接须在**最后工作日前完成**
- 交接完成后由主管签字确认

## 5. 离职结算

### 5.1 薪资结算

- 工资结算至**最后工作日**
- 结算在离职当月的**正常发薪日**发放
- 如最后工作日在发薪日之后，则在**次月发薪日**发放

### 5.2 年假折算

- 已享受年假超过折算应享天数的，**不扣回**
- 未休年假折算：
  - 折算天数 = 全年应享年假 × (当年已过日历天数 ÷ 365) - 已休天数
  - 折算金额 = 日工资 × 未休天数 × 200%（含已付的100%工资）

### 5.3 其他结算

| 项目 | 说明 |
|------|------|
| 加班调休 | 未使用的调休不折算现金，离职即清零 |
| 报销 | 离职前完成所有报销申请 |
| 借款 | 如有公司借款须在离职前归还 |
| 竞业限制补偿 | 如签署竞业协议，按约定按月发放 |

## 6. 社保与公积金

- 社保缴纳至**离职当月**（当月在职 15 天以上则缴纳当月社保）
- 公积金缴纳至**离职当月**
- 离职后社保/公积金转移由员工自行办理（或新单位接续）

## 7. 离职证明

- 最后工作日当天由 HR 出具
- 内容包含：在职起止日期、岗位、离职原因（个人原因/合同到期/协商解除）
- 如需多份，最多提供 **3 份**原件

## 8. 竞业限制

- 适用范围：核心技术人员、高级管理人员、其他约定人员
- 竞业期限：最长 **2 年**
- 竞业补偿：按月发放，标准为离职前12个月平均工资的 **30%**（不低于当地最低工资标准）
- 违约责任：以竞业协议约定为准

## 9. 离职面谈

- HR 将在员工最后一周安排离职面谈
- 面谈内容保密，不影响离职流程
- 员工可自愿参加
