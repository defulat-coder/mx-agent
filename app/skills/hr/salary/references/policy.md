# 薪酬福利制度

## 1. 薪资结构

| 组成部分 | 说明 |
|---------|------|
| 基本工资 | 根据职级确定，占月薪主体 |
| 绩效工资 | 根据季度/年度绩效考核结果浮动 |
| 岗位津贴 | 特殊岗位补贴（如技术津贴、管理津贴） |
| 加班费 | 按考勤制度中加班规则计算 |
| 专项补贴 | 餐补、交通补贴、通讯补贴等 |

## 2. 发薪规则

- **发薪日**：每月 **10 日**（遇周末/节假日提前至最近工作日）
- **发放方式**：银行转账至员工指定工资卡
- **工资条**：发薪日当天可在 HR 系统查看电子工资条
- **薪资周期**：每月 1 日至月末为一个计薪周期

## 3. 个人所得税

按国家累计预扣法计算：

### 3.1 税率表（综合所得）

| 级数 | 累计预扣预缴应纳税所得额 | 税率 | 速算扣除数 |
|------|----------------------|------|-----------|
| 1 | ≤ 36,000 | 3% | 0 |
| 2 | 36,000 - 144,000 | 10% | 2,520 |
| 3 | 144,000 - 300,000 | 20% | 16,920 |
| 4 | 300,000 - 420,000 | 25% | 31,920 |
| 5 | 420,000 - 660,000 | 30% | 52,920 |
| 6 | 660,000 - 960,000 | 35% | 85,920 |
| 7 | > 960,000 | 45% | 181,920 |

### 3.2 扣除项

- **起征点**：5,000 元/月
- **专项扣除**：社保 + 公积金个人部分
- **专项附加扣除**：子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人、婴幼儿照护（需员工在个税 APP 自行申报）

## 4. 年终奖

- **发放时间**：每年 **2 月**（春节前）
- **计算基数**：月基本工资 × 系数
- **绩效系数**：根据年度绩效评级确定

| 绩效评级 | 系数范围 |
|---------|---------|
| S（卓越） | 4-6 个月 |
| A（优秀） | 3-4 个月 |
| B+（良好） | 2-3 个月 |
| B（合格） | 1-2 个月 |
| C（待改进） | 0-1 个月 |

- 入职不满一年按实际在职月数折算
- 年终奖可选择单独计税或并入综合所得

## 5. 公司福利

### 5.1 补贴类

| 福利项目 | 标准 |
|---------|------|
| 餐补 | 25 元/工作日 |
| 交通补贴 | 500 元/月 |
| 通讯补贴 | 100-300 元/月（按职级） |
| 高温补贴 | 300 元/月（6-9 月） |

### 5.2 保障类

| 福利项目 | 说明 |
|---------|------|
| 补充医疗保险 | 公司全额购买，覆盖员工本人 |
| 年度体检 | 每年一次，公司指定体检机构 |
| 意外险 | 公司全额购买 |

### 5.3 其他福利

- 节日礼品/礼金（春节、端午、中秋等）
- 生日礼金：200 元
- 结婚礼金：1,000 元
- 生育礼金：1,000 元
- 丧葬慰问金：2,000 元
- 团建经费：200 元/人/季度
