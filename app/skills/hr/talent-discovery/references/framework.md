# 人才评估框架

## 评估维度

### 1. 绩效轨迹分析
- **持续优秀**：连续 2 期以上 A 或 B+，表明稳定的高产出能力
- **上升趋势**：评级逐期提升（如 B → B+ → A），表明成长潜力
- **停滞/下滑**：评级持平或下降，需关注原因

### 2. 学习与发展意愿
- **自主学习**：自主报名培训数量和完成率，反映内驱力
- **学习广度**：跨类别培训（如技术人员学管理课程），反映发展视野
- **IDP 执行力**：发展计划完成率和进度，反映目标导向能力

### 3. 职业成长速度
- **晋升频率**：入职至今的职级变动次数和间隔
- **职级停留时长**：当前职级停留超过 2 年需关注是否进入瓶颈期
- **跨部门经历**：调岗记录反映适应力和视野

### 4. 技能与能力
- **技能深度**：核心技能是否达到高级/专家水平
- **技能广度**：跨领域技能数量
- **认证资质**：外部认证证书数量和含金量

### 5. 项目贡献
- **项目角色**：负责人 > 核心成员 > 参与者
- **项目成果**：关键成果描述的质量和影响力
- **项目多样性**：参与不同类型项目的经验

## 分析标准

### 被埋没高潜识别
核心逻辑：**数据表现优秀但标签低估**
- 绩效连续 2 期 >= B+
- 自主报名培训 >= 3 门
- IDP 完成率 > 70%
- 当前九宫格标签为"普通"或位置为"中坚"及以下

### 流失风险预警
核心逻辑：**高价值 + 停滞信号**
- 绩效 A 或 B+（高价值确认）
- 当前职级停留 >= 2 年（停滞信号）
- 无活跃 IDP 或 IDP 已放弃（发展停滞）
- 加班趋势异常（可能是疲惫信号）

### 晋升准备度评估
综合评分维度及权重：
- 职级停留时长：25%（>= 2 年得满分，< 1 年低分）
- 最近绩效：30%（A=100, B+=80, B=60, C=30, D=0）
- 管理类培训：20%（完成数/目标数）
- IDP 完成度：25%（平均进度百分比）

### 岗位适配推荐
匹配优先级：
1. 技能标签直接匹配（最高权重）
2. 项目经历关键词匹配
3. 培训记录关键词匹配
4. 绩效基线过滤（>= B）

### 人才画像
全维度汇总，按以下顺序呈现：
1. 基本信息（姓名、部门、职级、司龄）
2. 教育背景
3. 核心技能
4. 项目成果
5. 绩效轨迹
6. 培训与发展
7. 九宫格定位
8. 职业发展建议

### 团队能力短板
分析维度：
- 技能覆盖度：每个技能的人数和等级分布
- 高频技能：团队 Top 5 技能
- 稀缺技能：仅 1 人掌握的技能（单点风险）
- 缺失技能：岗位要求但团队无人掌握

## 输出规范

分析报告应包含：
1. **数据支撑**：每个结论都要有具体数据佐证
2. **风险提示**：明确指出数据不完整对结论的影响
3. **可操作建议**：给出具体的下一步行动建议
4. **优先级排序**：多个候选人/风险时按优先级排序
