# 培训与职业发展制度

## 1. 培训体系

### 1.1 新员工培训

| 阶段 | 时间 | 内容 |
|------|------|------|
| 入职培训 | 入职第 1 天 | 公司介绍、制度宣讲、系统操作 |
| 岗位培训 | 入职第 1-2 周 | 业务知识、工具使用、流程规范 |
| 导师辅导 | 前 3 个月 | 一对一指导、定期 1v1 |

### 1.2 在职培训

| 类型 | 频率 | 说明 |
|------|------|------|
| 内部分享 | 每周/双周 | 部门内技术/业务分享 |
| 公司公开课 | 每月 | 跨部门知识分享 |
| 管理培训 | 按需 | 针对管理者的领导力课程 |
| 专项培训 | 按需 | 针对特定技能/工具的集中培训 |

### 1.3 外部培训

- 员工可申请外部培训/认证考试
- **审批流程**：员工申请 → 主管同意 → HR 审核 → 费用审批
- **费用承担**：
  - 公司指派：费用全额由公司承担
  - 个人申请：审批通过后公司承担，需签署培训服务期协议
- **服务期约定**：

| 培训费用 | 服务期 |
|---------|--------|
| ≤ 5,000 元 | 无 |
| 5,000 - 20,000 元 | 1 年 |
| 20,000 - 50,000 元 | 2 年 |
| > 50,000 元 | 3 年 |

## 2. 学习资源

| 资源 | 说明 |
|------|------|
| 内部知识库 | Confluence/飞书知识库，各团队沉淀的文档 |
| 在线学习平台 | 公司采购的课程账号（极客时间等） |
| 图书报销 | 每人每年 **500 元** 图书购买额度 |
| 技术社区 | 内部技术论坛，鼓励发帖与讨论 |

## 3. 职级体系

### 3.1 双通道

```
管理通道：M1 (主管) → M2 (经理) → M3 (高级经理) → M4 (总监) → M5 (VP)
                                                        
专业通道：P4 (初级) → P5 (中级) → P6 (高级) → P7 (专家) → P8 (资深专家) → P9 (首席)
```

- 两条通道可互转，需经评估
- P6 及以上可申请转管理通道
- 管理者也可申请回归专业通道

### 3.2 各级别能力要求概述

| 级别 | 能力要求 |
|------|---------|
| P4 | 在指导下完成模块级任务 |
| P5 | 独立完成项目级任务，能指导 P4 |
| P6 | 负责系统级设计与实现，具备跨团队协作能力 |
| P7 | 解决复杂问题，技术方案有行业影响力 |
| P8 | 定义技术方向，推动组织级技术演进 |
| P9 | 行业顶尖专家，引领技术战略 |

## 4. 晋升制度

### 4.1 晋升周期

- **常规晋升**：每年 **2 次**（4 月、10 月）
- **破格晋升**：无固定周期，需 VP 提名

### 4.2 晋升流程

```
主管提名（或自我申请）
  ↓
准备晋升材料（工作成果、项目案例、能力证明）
  ↓
晋升答辩（P6 及以上需要）
  ↓
评审委员会评审
  ↓
结果公示
  ↓
薪资调整（晋升次月生效）
```

### 4.3 晋升条件

| 条件 | 说明 |
|------|------|
| 任职时间 | 当前级别满 **1 年** 以上 |
| 绩效要求 | 最近 2 次绩效评级 B+ 及以上 |
| 能力匹配 | 达到目标级别的能力要求 |
| 主管推荐 | 直属主管书面推荐 |

### 4.4 晋升未通过

- HR 反馈改进方向
- 3 个月后可再次申请
- 连续 2 次未通过需与 HRBP 制定专项发展计划

## 5. 个人发展计划 (IDP)

- 每位员工每年初制定年度 IDP
- 内容包含：
  - 当前能力评估
  - 年度发展目标（1-3 个）
  - 学习计划与资源
  - 里程碑与检查点
- 与直属主管每季度 review 一次
- IDP 完成情况纳入年度绩效参考

## 6. 内部转岗

### 6.1 条件

- 在当前岗位满 **1 年**
- 最近一次绩效 B 及以上
- 直属主管知情同意

### 6.2 流程

```
员工申请（HR 系统）
  ↓
现主管审批（不可无故阻拦）
  ↓
目标部门面试
  ↓
HR 审核
  ↓
确定转岗日期（通常 1 个月交接期）
```

- 转岗后职级不变，薪资根据新岗位体系调整
- 转岗后有 **3 个月** 适应期
