# 马喜公司 IT 设备管理规范

## 一、设备配置标准

### 新员工入职标配
| 岗位类型 | 设备配置 |
|---------|---------|
| 开发/设计 | MacBook Pro 14" + 外接显示器 27" + 键鼠 |
| 产品/运营 | MacBook Air 13" 或 ThinkPad X1 Carbon |
| 行政/财务 | ThinkPad E14 + 外接显示器 24" |
| 管理层 | MacBook Pro 16" + 外接显示器 27" |

### 设备升级/更换
- 使用满 3 年可申请更换
- 设备性能明显不足可提前申请（需 IT 评估）
- 升级内存/硬盘需创建 IT 工单

## 二、设备领用流程

### 新员工入职
1. HR 发起入职流程后，IT 管理员自动收到设备分配通知
2. IT 管理员根据岗位标准准备设备
3. 员工入职当天到 IT 部门领取设备并签署《设备领用确认书》
4. IT 管理员协助完成系统安装和账号配置

### 领用确认
- 核对设备资产编号
- 检查设备外观是否完好
- 确认预装软件和账号均正常
- 签署领用确认书

## 三、设备归还流程

### 员工离职
1. HR 发起离职流程后，IT 管理员收到设备回收通知
2. 员工在最后工作日前将设备归还至 IT 部门
3. IT 管理员检查设备状态（外观、数据清除）
4. 双方签署《设备归还确认书》

### 归还要求
- 设备须恢复出厂设置或由 IT 清除数据
- 所有个人数据须提前备份（归还后不再提供）
- 配件须完整归还（充电器、键鼠、显示器等）
- 设备损坏须赔偿（按折旧价格计算）

## 四、设备借用规定

- 临时借用（如出差、培训）需创建 IT 工单并经主管审批
- 借用期限最长 30 天，可续借一次
- 借用设备须妥善保管，损坏或丢失需赔偿
- 归还时须由 IT 管理员检查确认

## 五、日常使用规范

### 必须做到
- 定期更新操作系统和安全补丁
- 保持杀毒软件开启并更新
- 离开工位锁定电脑
- 轻拿轻放，使用电脑包携带外出

### 禁止行为
- 私自拆装/改装硬件
- 使用设备从事与工作无关的活动
- 将设备借给非公司人员使用
- 在设备上安装非授权软件

## 六、设备报修

1. 遇到硬件故障（屏幕、键盘、电池等），通过智能助手创建报修工单
2. IT 管理员 4 小时内响应
3. 简单故障现场维修，复杂故障送修（提供备用机）
4. 保修期内免费维修，过保按实际费用处理
