# 公司邮箱配置与使用指南

## 一、邮箱基本信息

- 邮箱地址格式：`姓名拼音@maxi-tech.com`（如 `zhangsan@maxi-tech.com`）
- 邮箱服务：Microsoft 365
- 存储空间：50GB

## 二、Outlook 客户端配置

### Windows / macOS
1. 打开 Outlook → 「文件」→「添加账户」
2. 输入公司邮箱地址
3. 选择「Microsoft 365」→ 输入密码
4. 自动配置完成

### 手机端（iOS / Android）
1. 下载「Microsoft Outlook」App
2. 添加账户 → 输入公司邮箱
3. 选择「Office 365」认证
4. 输入密码并完成双因素认证

### 配置参数（手动配置时使用）
| 项目 | 值 |
|------|-----|
| 收件服务器 (IMAP) | `outlook.office365.com:993` |
| 发件服务器 (SMTP) | `smtp.office365.com:587` |
| 加密方式 | SSL/TLS |
| 认证方式 | OAuth 2.0 |

## 三、常见问题

### 1. 无法登录邮箱
1. 确认密码正确（与 AD 密码相同）
2. 检查密码是否过期（90 天强制更换）
3. 确认账号未被锁定（连续 5 次错误密码）
4. 尝试通过 Web 端登录：`https://outlook.office365.com`

### 2. 收不到邮件
1. 检查「垃圾邮件」文件夹
2. 确认邮箱容量未满（50GB 限制）
3. 检查邮件规则是否误设置了转发/删除
4. 让发件人确认发送状态（是否退信）

### 3. 发送失败
1. 检查收件人地址是否正确
2. 附件大小限制：单封邮件不超过 25MB
3. 群发上限：每天 10000 封
4. 检查是否被反垃圾邮件策略拦截

### 4. 邮箱容量满
1. 清理「已删除邮件」和「垃圾邮件」
2. 归档旧邮件到本地（Outlook → 「文件」→「归档」）
3. 删除大附件邮件（搜索 `size:>10MB`）

## 四、密码修改

- 公司内网：访问 `https://password.maxi-tech.com`
- 远程办公：先连 VPN，再访问上述地址
- 密码策略：最少 8 位，含大小写字母和数字，90 天更换一次
- 忘记密码：创建「密码重置」工单
