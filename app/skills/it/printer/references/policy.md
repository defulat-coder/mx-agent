## 二、打印机安装

### Windows
1. 打开「设置」→「设备」→「打印机和扫描仪」
2. 点击「添加打印机」→ 输入打印机 IP 地址
3. 系统自动搜索并安装驱动
4. 如驱动安装失败，从公司内网 `\\file-server\drivers\printer\` 下载

### macOS
1. 打开「系统偏好设置」→「打印机与扫描仪」
2. 点击「+」→「IP」选项卡
3. 输入打印机 IP 地址，协议选「AirPrint」
4. 系统自动识别打印机型号

## 三、常见问题排查

### 1. 无法打印
1. 确认打印机处于开机状态（指示灯正常）
2. 检查电脑与打印机是否在同一网段
3. 尝试重启打印服务（Windows: `services.msc` → Print Spooler → 重启）
4. 清空打印队列后重试
5. 删除打印机重新添加

### 2. 卡纸处理
1. 关闭打印机电源
2. 打开纸盒和后盖
3. 沿纸张输出方向缓慢拉出卡纸（避免撕裂）
4. 检查纸盒中纸张是否放置整齐
5. 重新开机测试

### 3. 打印质量差
1. **模糊/淡色**：碳粉不足，联系行政更换
2. **条纹/污点**：清洁硒鼓/转印带
3. **重影**：定影器老化，创建工单报修

### 4. 扫描/复印功能
1. 扫描到邮箱：在打印机面板选择「扫描到邮箱」，输入自己的公司邮箱
2. 扫描到 USB：插入 U 盘，选择「扫描到 USB」
3. 复印：放好原件 → 选择份数 → 按开始键

## 四、耗材更换

- 碳粉/墨盒更换：联系行政部门申领
- 纸张补充：各楼层打印机旁有纸张储物柜，自取即可
- 硒鼓/定影器等核心耗材：创建 IT 工单，由供应商处理
