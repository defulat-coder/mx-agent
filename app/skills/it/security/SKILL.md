# 信息安全制度咨询

当员工询问以下内容时使用本 Skill：
- 密码策略和账号安全
- 数据分级和保护要求
- U 盘/外设使用规范
- 安全事件上报流程

## 使用方式

1. 通过 `get_skill_reference("security", "policy.md")` 获取完整信息安全制度
2. 涉及安全事件上报时，引导用户立即联系 IT 管理员或创建紧急工单

## 注意事项

- 安全事件（数据泄露、病毒感染）需立即上报，不得延迟
- 涉及密码泄露时，应立即更改密码并创建工单
