# WiFi / VPN 连接排查指南

## 一、公司 WiFi 连接

### 网络名称
- 办公区：`MaXi-Office`（5GHz 优先）
- 访客区：`MaXi-Guest`（需前台获取临时密码）

### 常见问题排查

#### 1. 无法连接 WiFi
1. 确认已开启 WiFi 功能
2. 忘记已保存的网络，重新连接
3. 确认选择的是 `MaXi-Office` 而非 `MaXi-Guest`
4. 使用公司邮箱账号和密码认证（与 AD 账号相同）
5. 重启电脑后重试

#### 2. WiFi 连接后无法上网
1. 检查是否弹出认证页面（浏览器访问任意网页触发）
2. 尝试手动设置 DNS 为 `10.0.0.1`
3. 关闭 VPN 后测试（VPN 可能冲突）
4. 联系 IT 确认账号是否正常

#### 3. WiFi 频繁断开
1. 确认距离 AP 不太远（建议 15 米内）
2. 切换到 5GHz 频段
3. 关闭蓝牙排除干扰
4. 更新无线网卡驱动

## 二、VPN 连接

### 支持的 VPN 客户端
- Windows / macOS：FortiClient
- 手机端：FortiClient Mobile

### VPN 配置
- 服务器地址：`vpn.maxi-tech.com`
- 端口：`443`
- 认证方式：AD 账号 + 短信验证码

### 常见问题排查

#### 1. VPN 连接失败
1. 确认互联网连接正常
2. 检查 VPN 服务器地址和端口是否正确
3. 确认 AD 账号密码正确（与邮箱密码相同）
4. 检查短信验证码是否过期（有效期 60 秒）
5. 尝试切换网络（移动热点/家庭宽带）

#### 2. VPN 连接后网速慢
1. 断开 VPN 测试本地网速
2. 切换 VPN 线路（设置中选择"自动"）
3. 关闭不必要的大流量应用
4. 非工作时段使用分流模式

#### 3. VPN 账号锁定
- AD 账号连续 5 次密码错误将锁定 30 分钟
- 如需立即解锁，请联系 IT 管理员或创建工单

## 三、以上方法都无法解决

请通过智能助手创建 IT 工单，描述：
- 具体症状（截图最佳）
- 已尝试的排查步骤
- 设备型号和操作系统版本
