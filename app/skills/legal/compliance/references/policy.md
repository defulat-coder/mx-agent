# 马喜公司合规管理制度

## 一、行为准则

### 基本原则
1. 遵守国家法律法规和公司规章制度
2. 诚实守信，不得欺诈、隐瞒或虚假陈述
3. 维护公司利益，避免利益冲突
4. 尊重同事，营造健康工作环境

### 利益冲突
- 员工不得在与公司有业务往来的企业中持有重大股权
- 不得利用职务便利为自己或亲属谋取不正当利益
- 存在潜在利益冲突时必须向直属上级和合规部门报告

## 二、反腐败与反商业贿赂

### 禁止行为
1. 禁止向任何政府官员或商业伙伴提供贿赂
2. 禁止接受超过 200 元的礼品或招待
3. 禁止以公司名义进行任何形式的不正当利益输送
4. 禁止通过中间人或第三方进行间接贿赂

### 礼品与招待
| 场景 | 限额 | 审批要求 |
|------|------|---------|
| 接受礼品 | ≤ 200 元 | 无需审批 |
| 接受礼品 | > 200 元 | 须退还或上交公司 |
| 赠送礼品 | ≤ 500 元 | 部门负责人审批 |
| 赠送礼品 | > 500 元 | 合规部门审批 |
| 商务招待 | ≤ 500 元/次 | 部门负责人审批 |
| 商务招待 | > 500 元/次 | 分管副总审批 |

- 合规热线：400-XXX-XXXX
- 匿名举报系统：https://oa.maxi.com/compliance/report

## 三、数据隐私保护

### 个人信息保护
1. 收集员工/客户个人信息须明确告知用途
2. 个人信息存储须加密，访问须授权
3. 未经本人同意不得向第三方提供个人信息
4. 员工有权查询、更正、删除自己的个人信息

### 数据分级
| 级别 | 示例 | 保护措施 |
|------|------|---------|
| 公开 | 公司介绍、产品手册 | 无特殊要求 |
| 内部 | 内部通知、工作流程 | 仅限公司内部访问 |
| 机密 | 财务数据、客户信息 | 加密存储、权限控制 |
| 绝密 | 核心算法、战略规划 | 最小知悉原则、特殊加密 |

### 数据泄露处置
1. 发现数据泄露立即报告 IT 部门和合规部门
2. 评估泄露范围和影响
3. 采取补救措施（修改密码、撤销权限等）
4. 72 小时内向有关监管部门报告（如涉及个人信息）

## 四、审计配合

### 内部审计
- 每年至少一次全面内部审计
- 各部门须积极配合，提供所需文件和数据
- 审计发现问题须在 30 天内完成整改

### 外部审计
- 配合会计师事务所年度审计
- 如实提供财务数据和相关凭证
- 不得伪造、篡改或销毁审计相关资料

### 合规自查
- 各部门每季度进行一次合规自查
- 自查结果报送合规部门备案
- 自查发现问题及时整改并报告
