# 跨域协作场景 — Agent 评估用例

> 测试 Router Team 在用户请求涉及多个业务域时的协调能力
> 预期行为：Router 识别跨域意图 → 依次调度多个子 Agent → 汇总回复

---

## 一、新员工入职

| ID | 用户输入 | 预期调度顺序 | 验证点 |
|----|---------|------------|--------|
| CD-01 | "我是新入职的，需要办什么手续？" | HR → IT → Admin → Finance | 汇总包含入职手续+账号开通+工卡工位+报销账户 |
| CD-02 | "新员工第一天要做哪些事" | HR → IT → Admin | 至少涉及 HR 入职流程和 IT 设备领取 |
| CD-03 | "入职需要准备什么材料，怎么领电脑和工卡" | HR → IT → Admin | 识别出跨 HR + IT + 行政 |

---

## 二、员工离职

| ID | 用户输入 | 预期调度顺序 | 验证点 |
|----|---------|------------|--------|
| CD-04 | "我要离职，需要做什么交接" | HR → IT → Admin → Finance | 汇总包含离职流程+设备归还+工位清退+报销结算 |
| CD-05 | "离职需要归还哪些东西" | IT → Admin | 至少涉及设备归还和工卡退还 |
| CD-06 | "离职流程和经济补偿怎么算" | HR → Legal | HR 离职流程 + 法务经济补偿知识 |

## 三、出差全流程

## 四、混合咨询
